##Desciption
This project implements a comprehensive Information Retrieval (IR) system using the Cranfield dataset, covering the full pipeline from document processing to evaluation. The system performs text preprocessing through tokenization, stopword removal, and stemming to normalize both documents and queries. It constructs an inverted index that stores term frequencies and supports efficient retrieval. Query processing is performed using the same normalization steps to ensure consistency. Documents are ranked using multiple scoring approaches, including TF-IDF and the probabilistic BM25 model. The system also incorporates evaluation techniques to assess retrieval effectiveness, such as precision, recall, and ranking-based metrics, allowing comparison of different retrieval models. Overall, the project demonstrates the end-to-end design of a search system, including indexing, ranking, and evaluation, providing a practical understanding of modern information retrieval techniques.
